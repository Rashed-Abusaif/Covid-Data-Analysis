# Covid-Data-Analysis
Python-based analysis of U.S. COVID-19 data with trend visualizations and county-level choropleth mapping.
This project analyzes case and death data for a selected U.S. state using Python.

## Features
- Data cleaning and transformation using pandas
- Daily case and death calculations
- Yearly summaries (2020 and 2021)
- Data visualization:
  - Trend charts (cases and deaths)
  - County-level choropleth map

## Tools Used
- Python
- pandas
- geopandas
- matplotlib
- folium

## Example Output
- Total reported cases and deaths
- Average daily new cases and deaths
- Visual trends and geographic distribution
